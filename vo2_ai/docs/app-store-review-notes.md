## 4. Quick-reference answers to common reviewer questions

**Q: Is this app monetized in any way?**
No. It is free, open source, has no IAP, no subscriptions, no ads, no
affiliate links. Users bring their own OpenRouter key and pay OpenRouter
directly for LLM usage, not us.

**Q: Who is liable if a user gets injured following the AI plan?**
The app shows a clear medical disclaimer on first launch, which the user
must accept. The EULA (standard Apple EULA, Terms of Use link in app)
limits liability. The app never claims to be a medical device and does
not use medical terminology.

**Q: What data is sent off-device?**
Only what the user explicitly triggers:
  - Goal/level/preferences + (optional) Strava summaries → OpenRouter, under user's key
  - OAuth flow to Strava (if user taps Connect Strava)
Everything else is local (SQLite, iOS Keychain, UserDefaults).

**Q: Can the AI produce harmful content?**
The prompt is scoped tightly to running-training topics. Output is shown
only to the user who entered the prompt (no multi-user UGC, no sharing
features). Users can reset the chat at any time.
